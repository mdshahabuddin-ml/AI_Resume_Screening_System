# AI Resume Screening System

The **AI Resume Screening System** is a machine learning-based application designed to automate the process of analyzing and ranking candidate resumes. The system uses Natural Language Processing (NLP) and machine learning techniques to compare resumes with job descriptions and identify the most suitable candidates.

This project aims to assist recruiters and HR professionals by reducing the time required to manually review large numbers of resumes. The system extracts relevant information such as skills, experience, and keywords from resumes and evaluates how well they match the job requirements.

The project is divided into two main parts: a **backend** that handles machine learning processing and resume analysis, and a **frontend** that provides a simple interface for users to upload and manage resumes.

### Key Features

* Automatic resume text extraction
* Skill and keyword analysis using NLP
* Resume ranking based on job description matching
* Easy-to-use frontend interface
* Fast and efficient candidate screening

### Technologies Used

* Python
* Machine Learning
* Natural Language Processing (NLP)
* Pandas, NumPy, Scikit-learn
* HTML, CSS, JavaScript

### Project Structure

* **backend/** – Contains machine learning models and API logic
* **frontend/** – Contains the user interface for uploading resumes
* **requirements.txt** – Python dependencies for the project
* **render.yml** – Deployment configuration

### Purpose of the Project

The main objective of this project is to demonstrate how Artificial Intelligence can improve the recruitment process by automating resume screening and helping organizations find the best candidates more efficiently.
